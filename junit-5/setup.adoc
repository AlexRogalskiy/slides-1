
=== Running tests

With build tools:

* *Maven*:
** since Surefire 2.22.0
** add `test` dependency on engine
* *Gradle*:
** since 4.6
** add `testRuntime` dependency on engine
** activate JUnit 5:
+
[source,groovy]
----
test {
	useJUnitPlatform()
}
----

=== Running tests

As part of JUnit 4

* individual classes:
+
```java
@RunWith(JUnitPlatform.class)
public class JUnit5Test { ... }
```
* all classes:
+
```java
@RunWith(JUnitPlatform.class)
@SelectPackages({ "my.test.package" })
public class JUnit5TestSuite { }
```

=== Running tests

There is a console launcher:

```bash
# run all tests
java -jar junit-platform-console-standalone.jar
	--class-path ${path_to_compiled_test_classes}
	--scan-class-path
# run a specific test
java -jar junit-platform-console-standalone.jar
	--class-path ${path_to_compiled_test_classes}
	--select-class ${fully_qualified_test_class_name}
```
